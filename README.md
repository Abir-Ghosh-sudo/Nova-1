# 🚀 Nova-1 — Premium UI/UX Showcase

Nova-1 is a high-end frontend project that demonstrates modern UI/UX design, smooth animations, and interactive web experiences using pure HTML, CSS, and JavaScript.

It combines **two premium UI concepts** into a single project:

- 🛩️ Futuristic Jet Racing Experience (Quantara X-1)
- 💪 Modern Fitness Platform UI (Cult.fit Inspired)

---

## ✨ Features

- 🎨 Premium, modern UI design
- ⚡ Smooth animations & transitions
- 🧊 Glassmorphism & gradient effects
- 🧠 Advanced interactivity (modals, cursor, parallax)
- 📊 Dynamic UI elements & simulations
- 📱 Fully responsive (mobile + tablet + desktop)
- 🚀 No frameworks — built with pure HTML, CSS, JS

## 🖥️ Project Overview

### 🛩️ Futuristic Jet Racing UI (`index.html`)

A next-generation landing page for a fictional jet racing team featuring immersive visuals and advanced interactions.

#### 🔥 Key Features

- Custom animated cursor system  
- Parallax motion effects  
- Live telemetry simulation (dynamic data updates)  
- Interactive modals (Pre-order, Recruitment, Video)  
- Smooth scrolling navigation  
- Toast notifications  
- Section-based active navigation (Intersection Observer)  

#### 🎯 UI Sections

- Hero section with animated jet  
- Performance metrics (speed, thrust, acceleration)  
- Engineering features showcase  
- Team member cards  
- Interactive CTA & forms  

---

### 💪 Fitness UI Concept (`uiux.html`)

A clean, modern fitness landing page inspired by real-world platforms.

#### 🔥 Key Features

- Gradient-based modern design  
- Membership plans (Elite / Pro / Live)  
- Glassmorphism UI components  
- Floating animated elements  
- Responsive navigation menu  
- Stats and trust indicators  

#### 🎯 UI Sections

- Hero section with CTA  
- Membership pricing cards  
- Transformation banner  
- Stats section  
- Footer with links
